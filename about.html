About Weatherly

Weatherly is a simple and user-friendly weather web app that helps you track the current weather of any city in real-time. Built with modern web technologies, Weatherly provides essential weather information in a clean and accessible format.

Just type the name of a city, and Weatherly will fetch live data including temperature, humidity, wind speed, and a brief weather description.

Whether you're planning a trip, checking your hometown, or just curious about the weather somewhere else, Weatherly offers a quick and intuitive way to stay informed.

This project uses the OpenWeatherMap API to deliver reliable and accurate weather updates from around the world.
